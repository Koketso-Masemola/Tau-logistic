Tau Logistics Web Project
A professional, responsive website for Tau Logistics, an industrial supplier based in Ga-Masemola, South Africa. This project showcases industrial products, logistics services, and provides a streamlined enquiry system for clients.

Features
- Responsive Design: Optimized for mobile, tablet, and desktop views.
- Service Showcase: Detailed sections for industrial drums, chemical toilets, transport hire, and more.
- Lead Generation: Integrated enquiry and contact forms for business growth.


Project Structure
- index.html - The home page.
- about.html - Company history and mission.
- products.html - Full catalog of industrial supplies.
- enquiry.html - Booking and inquiry form.
- contact.html - Direct contact details and location.
- asset/ - Folder containing product images and icons.

 Technologie Used
- HTML5
